# Enterprise-Sales-Dataset
This is an Enterprise Sales Pipeline dataset basically a detailed record of how a sales team
operates. It tracks every deal, every customer interaction, every sales rep, and every client
company from 2022 to early 2026.
Think of it like the full history of a company's sales department stored in one file.

In undergoiong this project I crafted out 5 questions I would love to see answered in the project

## Project Deliverables:
- The Python file house the analysis
- Attached to it is an HTML report of the main findings
- Then a notebook in readable format (still under productin)
